## Descrição da Atividade

Esta atividade consiste na análise de um sistema real em produção por meio de **engenharia reversa baseada no comportamento**, sem acesso ao código-fonte. O sistema analisado é o cardápio digital da pizzaria **Tropykaly Pizzas e Lanches**, disponível em:

🔗 https://tropykalypizzaselanches.com.br/

## Objetivos

Aplicar na prática os conceitos de:
- Arquitetura de Software
- Design de Software
- Engenharia Reversa
- Modelagem UML
- Padrões de Projeto
